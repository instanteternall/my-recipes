# 我的菜谱项目启动说明

## 环境要求

### 后端
- Java 17+
- Maven 3.6+
- MySQL 8.0+

### 前端
- Node.js 16+
- npm 8+

## 启动步骤

### 1. 启动数据库（已配置）
- 数据库：MySQL
- 端口：3306
- 用户名：root
- 密码：123456
- 数据库名：my_recipes

### 2. 启动后端

```bash
cd backend

# 编译项目
mvn clean package -DskipTests

# 运行项目
java -jar target/my-recipes-backend-1.0.0.jar
```

后端默认端口：8080

### 3. 启动前端

```bash
cd frontend

# 安装依赖
npm install

# 运行项目
npm run dev
```

前端默认端口：3000

## 访问地址

- 前端：http://localhost:3000
- 后端API：http://localhost:8080
